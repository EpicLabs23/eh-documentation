# Install EHM

### Pre-requisite
- System Setup
- [EH Manager Installation](../eh-manager/eh-manager-instalation)
### Interactive Installation
```bash
eh-manager install
```

### Non-interactive Installation
```bash
eh-manager install -v v0.0.1 --dbpass drootp --apiurl http://localhost:2326 --os 24.04
```
