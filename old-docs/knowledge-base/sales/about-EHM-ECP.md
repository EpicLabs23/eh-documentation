# EHM & ECP

### What are they?
**EHM** stands for Epic Hosting Manager.

**ECP** Stands for Epic Control Panel.

Together we call them Epic Host ( **EH** ).

This is the most powerful shared hosting software. Installing this software in your server will turn your server into a shared hosting server. 

And one can manage entire server without any prior knowledge of complecated server technlogies like, Shell, User permission, Networking, DNS server, SSL certificate, virtual hosts, port mapping, file management, process management, email server, security and many more.

With the use of EHM and ECP, you can do it faster with confidence just by clicking on some visual buttons.

From EHM, you can manage main server and create user accounts. You may call EHM some sort of admin panel.

And the user accounts, created from EHM will access the system via a seperate panel, named ECP. You may call the ECP some some sort of customer panel.

### What is the use of a shared hosting server?

When you want to host multiple applications in a single server, so that you can save money. 

Or allow multiple people to use a single server, but keeping  them isolated from each other, so that one can't access another users' data or any kind of information.

For these situations, only perfect and secured solution is a shared hosting like Epic Host.


### Why Epic Host is the most powerfull?
This is the only shared hosting software that truely isolated users with very light weight yet secured virtualization technology.

EHM & ECP has support for all the popular languages, frameworks and  databases.

By achitecture, Epic Host can run any application. We are adding visual and other backend supports for more new type of applications.
